Bubbles game manual

Do you want to play Bubbles? Then here is how it works:

  • Start

    • It starts the game
    • It sends you to the first level

  • Settings

    • It sends you to the settings page where you can:
      • Modify the sound
      • Go back to the first page

  • Exit

    • You exit the game

To play the game you choose the odd one out based on the question given, and when you do you can:

  • Be right, and you go to the next level or you finish
  • Be wrong, and you go to the try again page where you can try again or go to the start page
